Ideabox

An exercise in learning how to create a basic Ruby/Sinatra app that used to form part of The Odin Project's curriculum (although it now appears to 404). Fortunately, the project is still available on the JumpstartLab website. This is one of the first functional "web apps" I've built, and I eventually hope to add more to it.

What is this?

This is a hybrid guided tutorial and independent work project with the goal of creating a basic web app which would allow the user to store and categorise their thoughts an ideas.

I really appreciated this approach to learning as it allowed for independent study and thought, which helped keep my attention long after most other tutorials or projects.
