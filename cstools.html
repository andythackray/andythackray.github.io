CS Tools updater

CS Tools is a Javascript based Userscript which was created by a co-worker in order to add some extra usability tweaks to our helpdesk software, admin panel and more. I contribute frequently, but due to the requirement to understand Javascript and HTML at a basic level in order to contribute, not all members of the team are able to make adjustments, which is problematic when the two that do understand are away.

What is this?

CS Tools Updater is a relatively basic Sinatra app that allows users to input and edit text using a WYSIWYG style interface, and then to update the Userscipt with that information, creating a download link for cstools.user.js which can be installed through extensions such as Tampermonkey. I created this due to the everchanging afersales information we received from rail operators in the early days of the COVID-19 outbreak. 

CS Tools supports a one click copy/paste feature, which copies a canned response. The canned response changes depending on the rail operator providing the journey, and also includes specifics about the journey/order item in order to personalise the message.

Previously, the only way to update the information in the canned responses was to manually alter the Userscript, which was daunting to less technical team members. Due to the high frequency of updates from rail operators, the script needed to be changed sometimes 3 times per day, which either reduced productivity of the two staff members able to make the updates, or was simply impossible due to forced staff absences.

By Creating CS Tools updater, I was able to make updates to the script much easier to manage for all users, allowing the tool be updated with the most recent information at the drop of a hat.

What now?

I learned a lot about deploying an app to Heroku, but I'm still not done.

For my next steps, I'm in the process of restructuring the database and then plan to convert the pages to Haml, before then look at adding additional features. Given the spike in work over the past few months due to the virus, I can't see this happening any time soon unfortunately.
